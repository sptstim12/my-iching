# 易经灵机 · 宣纸墨韵 UI 重构交付报告

 宣纸墨韵（淡雅中国风）

## 一、设计理念

**主轴：宣纸为底，水墨为骨，青绿点睛，朱砂提神。**

放弃原方案的「宫廷红+御金+夜黑」沉重调性，转向宋式美学与江南文人风。整体氛围从「庙堂庄严」转变为「清雅俊逸」，长时使用不疲劳。

| 维度 | 改造前 | 改造后 |
|------|--------|--------|
| 主背景 | 暗红黑渐变 #0d0505 | 宣纸米 #F4ECDD + 极淡纤维纹理 |
| 主文字 | 暗背景上的御金 | 浓墨 #2C2620 在宣纸上（对比度 11.8:1） |
| 点睛色 | 御金 #D4A017 | 朱砂 #B43A3A（仅用于强调） |
| 标题色 | 御金 + 投影 | 墨青 #2F4A4A（对比度 9.2:1） |
| 字体 | 系统楷体/宋体 | Google Fonts：Ma Shan Zheng + Noto Serif SC |
| 卡片底 | 宣纸贴暗背景 | 宣纸直接作为整页底，统一感强 |
| 签级标识 | 色块 | 朱砂方印（带白描边、-1° 倾斜） |

## 三、核心改动亮点

### 3.1 设计令牌系统（design-tokens.css）
建立完整的 token 体系，包括：
- **9 组色彩**（宣纸/墨/墨青/朱砂/御金/苔绿/签级/状态/阴影）
- **8 级字号**（11px → 40px）
- **12 级间距**（4px 基准：4/8/12/16/20/24/28/32/40/48/64/80）
- **4 种动效曲线 + 5 档时长**
- 完整支持 `prefers-reduced-motion`

### 3.2 视觉细节升级
- ✅ **印章式签级标签**：朱砂底 + 内白描边 + -1° 倾斜，更具古韵
- ✅ **"易"字方印**：Header 左上角朱砂小方印，呼应中国风
- ✅ **回纹角饰**：右上角自动翻转回纹（CSS transform 实现，无需新图）
- ✅ **变卦分隔小印**：居中"变"字朱砂方印
- ✅ **签诗/卦辞区**：渐变米色底 + 朱砂左竖线，圆角呼应卷轴
- ✅ **朱砂小标点**：标题前后的"·"、"◇"作为视觉断点

### 3.3 交互体验增强
- ✅ **首次引导浮层**：「静心·诚心·问心」渐进文字 + 入门按钮
- ✅ **Tab 切换动画**：淡出 + 上移 200ms 过渡
- ✅ **页面入场动画**：上浮渐显 500ms
- ✅ **结果卡进场动画**：上浮 16px 渐显 500ms
- ✅ **收藏功能**：localStorage 存储，按钮高亮态
- ✅ **再求一签**：直接重抽，不返回首页
- ✅ **Toast 提示**：统一的消息反馈
- ✅ **摇一摇优化**：仅在签诗页生效，避免误触
- ✅ **可访问性**：ARIA 标签、键盘 focus、reduced-motion

### 3.4 字体方案
```
标题:  Ma Shan Zheng（马善政体 - 飘逸书法，免费可商用）
正文:  Noto Serif SC（思源宋体 - 优雅宋体，免费可商用）
回退:  KaiTi / PingFang SC / Microsoft YaHei
```
